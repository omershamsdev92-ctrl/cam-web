# إعداد ميزة إرسال الرسائل النصية (SMS)

## الطريقة 1: إرسال SMS عبر السيرفر (Twilio) - موصى بها

### الخطوات:
1. إنشاء حساب مجاني في Twilio:
   - اذهب إلى: https://www.twilio.com/try-twilio
   - سجل حساب جديد (ستحصل على رصيد تجريبي مجاني)

2. الحصول على بيانات الاعتماد:
   - من لوحة التحكم، انسخ:
     * Account SID
     * Auth Token
   - من قسم Phone Numbers، احصل على رقم هاتف مجاني

3. إعداد المتغيرات البيئية:
   ```bash
   # انسخ ملف .env.example إلى .env
   cp .env.example .env
   
   # عدّل الملف وأضف بياناتك:
   TWILIO_ACCOUNT_SID=ACxxxxxxxxxxxxx
   TWILIO_AUTH_TOKEN=your_token_here
   TWILIO_PHONE=+1234567890
   ```

4. تثبيت المكتبات:
   ```bash
   npm install
   ```

5. إعادة تشغيل السيرفر:
   ```bash
   npm start
   ```

### المميزات:
✓ إرسال فعلي للرسائل
✓ يعمل حتى لو كان هاتف الضحية مغلقاً
✓ موثوق 100%
✓ يدعم أي رقم في العالم

## الطريقة 2: فتح تطبيق الرسائل في الهاتف (الطريقة الحالية)

إذا لم تقم بإعداد Twilio، سيستخدم النظام الطريقة البديلة:
- يفتح تطبيق الرسائل في هاتف الضحية
- الرسالة تكون جاهزة للإرسال
- يحتاج المستخدم للضغط على "إرسال" يدوياً
